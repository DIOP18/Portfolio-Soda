## À propos

Portfolio personnel de **Soda DIOP** - Développeuse full-stack, diplômée en Licence Génie Logiciel.

## Technologies utilisées

- React.js
- React-Bootstrap
- Node.js
- CSS3
- VsCode
- Vercel

## Fonctionnalités

**📖 Mise en page multi-pages**

**🎨 Stylisé avec React-Bootstrap et CSS**

**📱 Entièrement responsive**

## Installation

1. Clonez ce dépôt
2. Installez les dépendances : `npm install`
3. Lancez l'application : `npm start`

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Personnalisation

Ouvrez le dossier `/src/components/` pour modifier les informations du portfolio.
